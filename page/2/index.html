linux命令

  1. alias别名 防止删除错误 方便命令

  2. grep -n -a 2 “Failed” /var/log/message 打印上两行 -b打印下两行 -c打印上下两行 grep正则过滤

  3. sed 替换

  4. vim 高亮中的%s#master#node#g %表示全局，1表示第一行和sed差不多 替换 搜索

  5. ctrl + v 批量删除 替换

    • linux下的软件包类型有rpm简单需要依赖，版本偏低，源码开发的代码需要自己手动打包，二进制预制菜，解压即可使用，yum，最常用只需要有网自己解决依赖除了yum其他的都可能会有依赖centos他们的软件都是rpm结尾的软件包

    • [rpm包地址](Rpmfind mirror)

  7. is-enable 开机是否自启 systemd软件管理

  8. journalctl -xe 是可以进行搜索的

  9. zabbix-release（包含repo文件）

  10. 磁盘满后用 df -sh /* 精确到大文件，确认后删除

  11. 没有包只知道命令可以用yum provides pstree来查询这个包的名称，pstree是查看进程树的 pstree -p查看进程号

linux快捷键

1.通用快捷键

  • Ctrl + A 或 Home：将光标移动到行首。

  • Ctrl + E 或 End：将光标移动到行尾。

  • Ctrl + B 或 Left：向左移动光标一个字符。

  • Ctrl + F 或 Right：向右移动光标一个字符。

  • Ctrl + K 或 Ctrl + U：删除光标后的所有字符/剪切光标前的字

2.文件内快捷键

  • G到文件末尾

  • gg到文件开头

  • d0删除光标前一个字母到首行

  • D删除光标整行到末尾

  • X删除光标前的字符

  • x删除光标后的字符
